자바스크립트의 표준 데이터형 객체

Number 객체

생성
var 객체명 = New Number(숫자값);

관련 속성과 메서드

  • 속성
  • Number.MAX_VALUE : 자바스크립트에서 사용 가능한 가장 큰 숫자를 반환(자바스크립트 中 Number에만 내장된 속성)(변환불가)
  • Number.MIN_VALUE : 자바스크립트에서 사용 가능한 가장 작은 숫자를 반환(자바스크립트 中 Number에만 내장된 속성)(변환불가)
  • Number.POSITIVE_INFINITY : 양의 무한대 Infinity를 반환
  • Number.NEGATIVE_INFINITY : 음의 무한대 Infinity를 반환
  • Number.NaN : 숫자가 아님(Not-a-Number)을 나타내는 숫자값
  • 메서드
    • //"객체."의 경우 지정해준 객체를 입력하고, "Number"의 경우는 Number만 입력이 가능하다.
  • Number.isFinite(변수 또는 값) : 유한수인지 아닌지 반환, 유한수이면 true
    • //is는 "~이다"의 뜻으로 여기서는 정보함수로 사용한다.
  • Number.isInterger(변수 또는 값) : 정수인지 아닌지 반환, 정수이면 true
    • //is는 "~이다"의 뜻으로 여기서는 정보함수로 사용한다.
  • Number.isSafeInterger(변수 또는 값) : 변수에 담긴 값이 안전한 정수값인지 아닌지 반환, 안전한 정수이면 true
    • //is는 "~이다"의 뜻으로 여기서는 정보함수로 사용한다.
  • 객체.toExponential(유효숫자자릿수) : 지수형태의 문자열로 가공하여 반환
    • //to는 가공하여 값을 변환,
  • 객체.toFixed(소숫점이하자릿수) : 지정한 자릿수 만큼 반올림하여 반환
    • //to는 가공하여 값을 변환
  • 객체.toPrecision(반올림할유효숫자자릿수) : 매개변수로 지정된 정체 자릿수까지 유효하도록 나머지 자릿수를 반올림하여 문자열로 변환
    • //to는 가공하여 값을 변환
  • 객체.toString(진수) : 숫자를 원하는 진수의 문자열로 변환
    • //to는 가공하여 값을 변환
