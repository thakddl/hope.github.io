자바스크립트의 표준 데이터형 객체

String 객체

변수 또는 객체에 관계없이 모두 사용 가능

관련 메서드

  • 속성
  • 객체.length : 문자열을 글자수를 보관
  • 메서드
  • 객체.charAt(인덱스번호) : 지정한 인덱스번호에 해당되는 문자를 반환
  • 객체.charCodeAt(인덱스번호) : 지정한 인덱스번호에 해당되는 문자를 아스키코드로 반환
  • String.fromeCharCode(아스키코드번호) : 지정한 아스키코드번호에 해당되는 문자로 반환
  • 객체.concat(문자열1, 문자열2....) : 객체의 문자열값과 괄호 안의 문자열 값들을 더한 결과 반환
  • 객체.indexOf(특정문자또는단어) : 객체에서 특정문자나 단어의 맨 처음 찾은 위치를 인덱스 번호로 반환
  • 객체.lastIndexOf(특정문자또는단어) : 객체에서 특정문자나 단어의 맨 나중에 찾은 위치를 인덱스번호로 반환
  • 객체.replace(찾을단어, 바꿀단어) : 객체의 문자열에서 찾을단어를 찾아 바꿀단어로 변경하여 반환
  • 객체.split(구분기호) : 객체의 문자열을 지정한 구분기호로 분리하여 배열형태로 반환
  • 객체.substr(시작인텍스번호[ ,글자수]) : 객체에서 시작인덱스번호에 속하는 문자부터 지정한
  • 객체.substring(시작인덱스번호[ ,글자수]) : 객체에서 시작인덱스번호에 속하는 문자부터 끝인덱스번호 전까지를 추출하여 반환 //음수 사용 불가능
  • 객체.slice(시작인덱스번호[ ,끝인덱스번호]) : 객체에서 시작인덱스번호에 속하는 문자부터 끝인덱스번호 전까지를 잘라내어 반환(음수일 경우 뒤에서 부터) //음수 사용 가능
  • 객체.toLowerCase() : 객체의 문자열값을 모두 소문자로 변환하여 반환
  • 객체.toUpperCase() : 객체의 문자열값을 모두 대문자로 변환하여 반환
  • 객체.trim() : 객체의 양쪽 끝에 있는 공백을 제거하여 반환
  • 객체.repeat(반복횟수) : 객체의 문자열을 반복횟수 만큼 반복하여 반환
  • 객체.includes(문자열) : 객체에 해당 문자열이 있는지 검색하여 있으면 true 반환
  • 객체.macth(비교할문자열) : 객체에 해당 문자열이 있으면 찾은 문자열을 배열로 반환
  • 객체.matchAll(비교할문자열패턴) : 객체에 해당 문자열이 포함되어 있으면 찾은 문자열을 가장 비교할문자열패턴과 일치하는 것부터 배열로 반환
  • 객체.valueOf() : 무자열형 객체를 일반 변수로 반환
