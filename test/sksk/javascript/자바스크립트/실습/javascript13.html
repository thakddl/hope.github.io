자바스크립트의 표준 데이터형 객체

Math 객체

생성자 없이 사용

관련 속성과 메서드

  • 속성
  • Math.E : 오일러의 상소이며 자연로그의 밑, 약 2.718
  • Math.LN2 : 2의 자연로그, 약 0.693
  • Math.LN10 : 10의 자연로그, 약 2.303
  • Math.LN2E : 밑이 2인 로그 E, 약 1.443
  • Math.LOG2E : 밑이 10인 로그 E, 약 0.434
  • Math.PI : 원의 둘레와 지름의 비율, 약 3.14159
  • Math.SQRT1_2 : 1/2의 제곱근, 약 0.707
  • Math.SQRT2 : 2의 제곱근, 약 1.414
  • 메서드
  • Math.abs(변수/연산식/값) : 괄호 안의 결과값을 절대값으로 반환
  • Math.round(변수/연산식/값) : 괄호 안의 결과값을 소숫점 이하 반올림한 정수로 반환
  • Math.ceil(변수/연산식/값) : 괄호 안의 결과값을 소수점 이하 올림한 정수로 반환
  • Math.floor(변수/연산식/값) : 괄호 안의 결과값을 소수점 이하 내림한 정수로 반환
  • Math.sqrt(변수/연산식/값) : 괄호 안의 결과값에 대한 제곱근을 반환
  • Math.random(변수/연산식/값) : 괄호 안의 결과값에 대한 임의의 수를 반환
  • Math.pow(자연수, 승수) : 괄호 안의 자연수에 대한 승수에 결과를 반환
  • Math.max(숫자1, 숫자2...) : 괄호 안의 숫자 분포에서 가장 큰 수를 반환
  • Math.min(숫자1, 숫자2...) : 괄호 안의 숫자 분포에서 가장 작은 수를 반환
  • Math.sin(각도숫자) : 괄호 안의 각도숫자에 대한 sin 값을 구하여 반환
  • Math.cos(각도숫자) : 괄호 안의 각도숫자에 대한 cos 값을 구하여 반환
  • Math.tan(각도숫자) : 괄호 안의 각도숫자에 대한 tan 값을 구하여 반환

Math가 생략되어 사용하는 범용 메서드

  • parseInt(변경할 값) : 정수로 변경시 사용
  • parseFloat(변경할 값) : 실수로 변경시 사용
  • Number(변경할 값) : 숫자로 변경시 사용
