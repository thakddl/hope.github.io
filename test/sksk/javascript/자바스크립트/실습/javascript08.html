조건문

if문

주어진 조건이 만족할 때와 만족하지 않을 경우에 따라 실행할 내용을 달리함 -> 분기문

문법1 : if(조건) 조건이 참일때 처리문장;

문법2 : if(조건) { 조건이 참일 때 처리문장; }

문법3 : if(조건) { 조건이 참일 때 처리문장; } else {조건이 거짓일 때 처리문장; }

문법4 : if(조건) { 조건이 참일 때 처리문장; } else if(조건2) { 조건2가 참일 때 처리문장;} ... else { 주어진 조건이 무두 거짓 일떄 처리 문장}



switch문

주어진 기준변수의 값이 무엇이냐에 따라 실행할 내용을 달리함 -> 선택문

문법
case 값1;
기준변수의 값이 값1에 해당하는 경우 처리문장;
break;
case 값2;
기준변수의 값이 값2에 해당하는 경우 처리문장;
break; ......
> default: 기준변수의 값이 위 case 중 만족된 것이 없는 경우 처리문장;
