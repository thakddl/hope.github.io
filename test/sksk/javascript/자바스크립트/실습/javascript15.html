자바스크립트의 표준 데이터형 객체

Array 객체

변수 또는 객체에 관계없이 모두 사용 가능

관련 속성과 메서드

  • 속성
  • 객체.length : 배열의 요소의 수를 보관
  • 메서드
  • Array.isArray(배열변수또는객체) : 변수 또는 객체가 배열인지 확인
  • Array.from(문자열) : 문자열을 배열로 만듦
  • Array.of(요소1,요소2,...요소n) : 지정한 요소 만큼 배열로 만듦
  • 객체.indexOf(찾을값) : 배열객체에 찾을값에 해당하는 요소가 있는지 비교
  • 객체.concat(배열) : 두 배열을 합친 배열을 만듭니다.
  • 객체.join(문자) : 지정한 문자로 요소를 구분자로하여 출력
  • 객체.pop() : 맨 끝 요소를 제거
  • 객체.push(요소값) : 요소값을 맨 끝 요소를 추가
  • 객체.shift() : 맨 처음 요소를 제거
  • 객체.unshift(요소값) : 요소값을 맨 처음 요소로 추가
  • 객체.slice(처음인덱스,끝인덱스) : 처음인덱스부터 끝인덱스 전까지를 배열로 추출하여 복사된 배열로 만듦
  • 객체.splice(처음인덱스, 개수) : 처음익덱스부터 개수만큼 애려로 추출하여 이동된 배열로 만듦
  • 객체.reverse() : 배열요소의 순서를 뒤집습니다
  • 객체.sort() : 배열요소를 오름차순 순서로 정렬
