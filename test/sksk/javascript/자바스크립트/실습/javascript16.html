자바스크립트의 표준 데이터형 객체

1998-12-16

Date 객체

시간객체에만 사용 가능함

생성자함수(Constructor) : new Date(year, month[, day, hour, minute, second, millisecond])

생성자 함수에 의해서만 활요 가능함

관련 속성과 메서드

  • 메서드
  • Date.now() : 1970년 1월 1일 00:00:00(UTC)을 기점으로 현재 시간까지 경과한 밀리초를 숫자로 반환
  • Date.parse("년월일시분초") : 1998년 1월 1일 00:00:00:(UTC)을 기점으로 인수로 전달된 지정 시간(new Date(dateString)의 인수와 동일한 형식)까지의 밀리초를 숫자로 반환
  • 객체.getFullYear() : 년도를 나타내는 4자리 숫자를 구함
    • //get : 값을 가져오다
  • 객체.setFullYear() : 날짜 객체에 지정한 4자리 숫자값을 년도로 설정
    • //set : 값을 설정하다
  • 객체.getYear() : 년도를 나타내는 2자리 숫자를 구함
  • 객체.setYear(년도) : 날짜 객체에 지정한 2자리 숫자값을 년도로 설정
  • 객체.getMonth() : 월을 나타내는 숫자를 구함(단, 0~11)
  • 객체.setMonth(월) : 날짜 객체에 지정한 0~11의 숫자값을 월로 설정
  • 객체.getDate() : 일을 나타내는 숫자를 구함
  • 객체.setDate(일) : 날짜 객체에 지정한 숫자값을 일로 설정
  • 객체.getDay() : 요일번호를 나타내는 숫자를 구함(0~6)
  • 객체.getHours() : 시를 나타내는 숫자를 구함
  • 객체.setHours(시) : 날짜 객체에 지저한 숫자값을 시로 설정
  • 객체.getMinutese() : 분을 나타내는 숫자를 구함
  • 객체.setMinutes(분) : 날짜 객체에 지정한 숫자값을 분으로 설정
  • 객체.getSeconds() : 초를 나타내는 숫자를 구함
  • 객체.setSeconds(초) : 날짜 객체에 지정한 숫자값을 초로 설정
  • 객체.getMilliseconds() : 밀리초를 나타내는 숫자를 구함
  • 객체.setMilliseconds(초) : 날짜 객체에 지정한 숫자값을 밀리초로 설정
  • 객체.getTime() : 1970년 1월 1일 00:00:00(UTC)를 기점으로 현재 시간까지 경과된 밀리초를 설정
  • 객체.getTimezoneOffset() : UTC와 지정 로케일(Locale) 시간과의 차이를 분단위로 반환
  • 객체.toDateString() : 사람이 읽을 수 있는 형식의 문자열로 날짜를 반환
  • 객체.toTimeString() : 사람이 읽을 수 있는 형식의 문자열로 시간을 반환
