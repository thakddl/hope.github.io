반복문

for문

순서값을 증감시켜 조건이 만족하는 동안 원하는 만큼 처리문장을 반복

문법1
for(변수설정 및 초기화;조건;증감식) {
반복처리할문장;
}

문법2
for(var key in 배열변수및객체명) {
반복처리할문장;
}

문법3
배열변수명.forEach(function(element, index) {
반복처리할문장;
});



while문

조건이 만족하는 동안 블록처리문장을 반복 수행

문법1
while(조건) { 반복처리할문장; }

문법2
do{ 반복처리할문장; } while(조건);



무한반복(infinite loop)문장

for문이나 while문이 실행에 끝이 없이 반복 수행

문법1
for(;;){ 반복처리할문장; }

문법2
while(){반복처리할문장;}

조건이 항상 만족하거나 생략된 경우 무한반복문장이 됨



반복문의 제어

continue문

실행구간을 건너띄도록 하는 제어문

문법
for(변수설정및초기화;조건;증감식){ if(조건) continue; }

break문

더 이상 실행되지 못하도록 하는 제어문

문법
for(변수설정및초기화;조건;증감식){ if(조건) break; }
