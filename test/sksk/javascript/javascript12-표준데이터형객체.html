자바스크립트의 표준 데이터형 객체

Number객체

생성
var 객체명 = new Number(숫자값);

관련 속성과 메서드

//자바스크립트의 내장 함수이다.
  • 속성
  • 객체.MAX_VALUE : 자바스크립트에서 사용 가능한 가장 큰 숫자를 반환
  • 객체.MIN_VALUE : 자바스크립트에서 사용 가능한 가장 작은 숫자를 반환
  • 객체.POSITIVE_INFINITY : 양의 무한대 Infinity를 반환
  • 객체.NEGATIVE_INFINITY : 음의 무한대 Infinity를 반환
  • 객체.NaN : 숫자가 아님(Not-a-Number)을 나타내는 숫자값
  • 메서드
    • //변수 또는 값이 괄호 안에 들어가야한다는 것을 잊지말자
  • Number.isFinite(변수 또는 값) : 유한수인지 아닌지 반환. 유한수이면 true
  • Number.isInteger(변수 또는 값) : 정수인지 아닌지 반환. 정수이면 true
  • Number.isNaN(변수 또는 값) : 계산이 가능한 숫자인지 아닌지 반환. 숫자가 아니면 true
  • Number.isSafeInteger(변수 또는 값) : 정수에 담긴 값이 안전한 정수값인지 아닌지 반환. 안전한 정수이면 true
  • 객체.toExponential(유효숫자 자릿수) : 지수형태의 문자열로 가공하여 반환
  • 객체.toFixed(소숫점 이하 자릿수) : 지정한 자릿수 만큼 반올림하여 반환
  • 객체.toPrecision(반올림할 유효숫자 자릿수) : 매개변수로 지정된 전체자릿수까지 유효하도록 나머지 자릿수를 반올림하여 문자열로 반환
  • 객체.toString(진수) : 숫자를 원하는 문자열로 반환
